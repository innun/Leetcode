# 109. 有序链表转换二叉搜索树

### 描述：
给定一个单链表，其中的元素按升序排序，将其转换为高度平衡的二叉搜索树。
本题中，一个高度平衡二叉树是指一个二叉树每个节点 的左右两个子树的高度差的绝对值不超过 1。
### 示例：
```
给定的有序链表： [-10, -3, 0, 5, 9],

一个可能的答案是：[0, -3, 9, -10, null, 5], 它可以表示下面这个高度平衡二叉搜索树：

      0
     / \
   -3   9
   /   /
 -10  5
```
### 解答：
```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Solution {
    public TreeNode sortedListToBST(ListNode head) {
        ArrayList<Integer> a = new ArrayList<>();
        listToArray(head, a);
        if (a.size() == 0) {
            return null;
        }
        return sortedListToBST(a, 0, a.size() - 1);
    }

    void listToArray(ListNode head, ArrayList<Integer> a) {
        while (head != null) {
            a.add(head.val);
            head = head.next;
        }
    }

    TreeNode sortedListToBST(ArrayList<Integer> a, int lo, int hi) {
        if (lo > hi) {
            return null;
        }
        int mid = (lo + hi) / 2;
        TreeNode node = new TreeNode(a.get(mid));
        node.left = sortedListToBST(a, lo, mid - 1);
        node.right = sortedListToBST(a, mid + 1, hi);
        return node;
    }
}
```
