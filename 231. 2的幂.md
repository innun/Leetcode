# 231. 2的幂

### 描述：
给定一个整数，编写一个函数来判断它是否是 2 的幂次方。
### 示例1：
```
输入: 1
输出: true
解释: 2^0 = 1
```
### 示例2：
```
输入: 16
输出: true
解释: 2^4 = 16
```
### 示例3：
```
输入: 218
输出: false
```
### 解答：
假设x为2的幂次方，其二进制形式只有最高位为1，因此x与x-1进行与操作得到的值必为0
```java
class Solution {
    public boolean isPowerOfTwo(int n) {
        if (n <= 0) {
            return false;
        }
        return (n & (n - 1)) == 0;
    }
}
```
