# 338. 比特位计数

### 描述：
给定一个非负整数 num。对于 0 ≤ i ≤ num 范围中的每个数字 i ，计算其二进制数中的 1 的数目并将它们作为数组返回。
### 示例1：
```
输入: 2
输出: [0,1,1]
```
### 示例2：
```
输入: 5
输出: [0,1,1,2,1,2]
```
### 解答：
偶数比特位为1的个数和其右移一位得到的数的比特位为1的个数相同
奇数比特位为1的个数比其右移一位得到的数的比特位为1的个数多1
```java
class Solution {
    public int[] countBits(int num) {
        int[] res = new int[num + 1];
        for (int i = 1; i < num + 1; i++) {
            if ((i & 1) == 0) {  //若i为偶数
                res[i] = res[i >> 1];
            } else {             //若i为奇数
                res[i] = res[i >> 1] + 1;
            }
        }
        return res;
    }
}
```
